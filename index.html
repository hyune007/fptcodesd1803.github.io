    • Học để lập trình
    Với trang web học lập trình lớn nhất Việt Nam
    Nếu bạn không thực sự biết nên bắt đầu từ đâu

    HTML

    Ngôn ngữ xây dựng trang web

    HTML Example:

    
                            
    
        <!DOCTYPE html>
    
        <html>
    <head>
    
        <title>HTML Tutorial</title>
    </head>
    
        <body>

    
        <h1>This is a heading</h1>
    
        <p>This is a paragraph.</p>

    
        </body>
    
        </html>

    CSS

    Ngôn ngữ tạo kiểu cho trang web

    CSS Example:

    
    body {
    background-color: lightblue;
    }
    
    h1 {
    color: white;
    text-align: center;
    }
    
    p {
    font-family: verdana;
    }

    JavaScript

    Ngôn ngữ lập trình trang web

    CSS Example:

    
    <button onclick="myFunction()">Click Me!</button>

    
    <script>
    
    function myFunction() {
    
       let x = document.getElementById("demo");
    
      x.style.fontSize = "25px"; 
    
      x.style.color = "red"; 
    }
    
        </script>
